CSS浮动(Float)语法教程

学习如何使用float属性创建布局并了解其特性

Float语法

float: left | right | none | inherit;

属性值说明

  • left - 元素浮动到其容器的左侧
  • right - 元素浮动到其容器的右侧
  • none - 默认值，元素不浮动
  • inherit - 元素继承其父元素的float值

清除浮动

使用clear属性可以指定元素的哪一侧不允许出现浮动元素：

clear: left | right | both | none;

清除浮动技巧（Clearfix）

当容器内的所有元素都浮动时，容器的高度会塌陷，可以使用clearfix技巧解决：

/* 现代clearfix hack */
.clearfix::after {
    content: "";
    display: table;
    clear: both;
}
