  • 圣杯布局

布局技术

现代网页布局主要使用以下技术：

  • Flexbox - 一维布局模型
  • CSS Grid - 二维布局系统
  • Float - 传统布局方法（仍有用武之地）
  • Positioning - 精确控制元素位置
/* 使用Flexbox创建两列布局 */
.container {
    display: flex;
}
.main {
    flex: 3;
}
.sidebar {
    flex: 1;
}

响应式设计

现代网页布局必须是响应式的，能够适应不同屏幕尺寸。使用CSS媒体查询可以实现这一点：

/* 移动设备样式 */
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
}
